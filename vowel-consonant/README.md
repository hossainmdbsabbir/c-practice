# Vowel or Consonant Checker (C)

This program checks whether a given character is a vowel or a consonant through if else and switch case.

## Concepts Used
- If-else condition
- Switch-case condition
- Character comparison

## How to Run
1. Compile the program
2. Enter a character
3. The program displays whether it is a vowel or consonant

## Purpose
Practice character-based conditions in C.
