# Largest/Smallest of Three Numbers (C)

This program finds the largest/smallest number among three numbers using if-else ladder and nested conditions.

## Concepts Used
- Nested if-else
- Ladder if-else
- Comparison operators

## How to Run
1. Compile the program
2. Enter three numbers
3. The largest/smallest number is displayed

## Purpose
Practice decision-making logic in C through nested and ladder if else.
